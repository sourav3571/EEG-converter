# EEG Imagined Spanish Sentence Decoder

An interactive Streamlit dashboard and deep learning model decoder for Spanish imagined speech based on the ds004279 dataset.

## Installation & Running Locally

1. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
2. Run Streamlit:
   ```bash
   streamlit run eeg_dashboard/app.py
   ```
